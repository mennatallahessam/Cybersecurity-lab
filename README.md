# Cybersecurity Home Lab

Welcome to my Cybersecurity Home Lab repository! This repository serves as my personal portfolio project showcasing hands-on experience with cybersecurity concepts, Linux system administration, and security best practices. Each assignment represents a focused exploration into different aspects of cybersecurity through practical, real-world scenarios conducted on a virtual machine.

## About This Lab

This home lab is designed to build a strong foundation in:
- **Linux System Administration** - User management, permissions, and access control
- **Security Fundamentals** - Best practices for system hardening and access control
- **Command Line Proficiency** - Essential skills for security professionals
- **Hands-On Learning** - Practical experience with real security concepts

Each chapter documents the commands used, their purpose, and the outcomes achieved, providing a comprehensive learning record and professional portfolio showcase.

## Chapters

### [Assignment 2 - Linux Security Basics Lab](./Assignment%202%20Linux%20Security%20Basics/README.md)
**Topics Covered:**
- System updates and maintenance
- User account management (useradd vs adduser)
- Privilege escalation and sudo configuration
- Group management and membership
- File permissions and Access Control Lists (ACLs)
- Bash scripting basics

This assignment explores fundamental Linux security concepts including user and group management, permission models, and access control. Through hands-on terminal commands, we examine how to properly configure user privileges, manage file permissions, and implement security best practices on a Linux system.
